MTMCookieName

Possible values: The name as a string
Default value: Not set
Requirements: MTMUseCookies = true;
First available: 2.3.0
Supported browsers: All
Description: This option specifies the name of the cookie where the menu expansion state is stored when using cookies. It can be set to any value of choice. Keep in mind that it could be a good idea to change this value if you change your menu, to make sure that the expansion state is correctly stored.
