MTMUseToolTips

Possible values: true or false
Default value: true
Requirements: none
First available: 2.3.0
Supported browsers: All but NN3 & NN4
Description: This controls the usage of tool tips in the menu, which is done by inserting each item's tool tip text into the title -attribute of the links. In other words this in straight HTML and requires support for that attribute in the browser. Since Netscape Navigator 3 & 4 do not support it they won't support the tool tips.
