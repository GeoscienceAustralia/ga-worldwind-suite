MTMAlwaysLinkIfWE

Possible values: true or false
Default value: true
Requirements: MTMEmulateWE = true;
First available: 2.3.1
Supported browsers: All
Description: This variable controls the display of submenus when MTMEmulateWE is set to true. The default value of MTMAlwaysLinkIfWE is true, which means that the whole submenu is a link that expands & collapses the submenu. This means the +/- sign, the icon and the text. If you set it to false only the +/- sign and the icon will expand and collapse the submenu, and the text will only be text. If you set it to false you will also have to set MTMSubTextColor to the right color for your site.
