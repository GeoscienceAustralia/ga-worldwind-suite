MTMrightClickMessage

Possible values: A message text as a string
Default value: Not set
Requirements: none
First available: 2.3.1
Supported browsers: IE4
Description: This option exists only to enable prevention of right-clicking the mouse in the menu frame for users with Microsoft Interexplorer 4.0. By setting this option the defined text will be inserted into the menu frame together with a right-click prevention script. It does not work in any other browsers, so if you want a feature like this you should also look into the usage of MTMLinkedJSURL and MTMLinkedInitFunction.
