MTMTimeOut

Possible values: A number, larger than or equal to 0
Default value: 5
Requirements: none
First available: 2.2.2
Supported browsers: All
Description: This specifies a timeout period in tenths of a second (1/10), the default value of 5 therefore represents half a second. The value is only used when your site is being loaded, or when the frameset (and thereby code.html) is reloaded. If a document in the frameset doesn't contain the tracking code, or said code doesn't update the menu before the specified amount of time, there will be a forced display/update of the menu. This is to make sure the menu is shown to the user.
