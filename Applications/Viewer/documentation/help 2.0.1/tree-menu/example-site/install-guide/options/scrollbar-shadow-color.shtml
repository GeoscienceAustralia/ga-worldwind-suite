MTMscrollbarShadowColor

Possible values: A valid CSS colour value as a string
Default value: none
Requirements: MTMLinkedSS = false;
MTMuseScrollbarCSS = true;
First available: 2.3.1
Supported browsers: IE5.5, IE6
Description: This option controls the colour of the bottom and right edges of the scroll box and scroll arrows of the scroll bar in the menu frame.
