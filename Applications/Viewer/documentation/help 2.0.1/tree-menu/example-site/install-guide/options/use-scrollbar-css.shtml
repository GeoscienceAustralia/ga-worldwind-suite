MTMuseScrollbarCSS

Possible values: true or false
Default value: false
Requirements: MTMLinkedSS = false;
First available: 2.3.1
Supported browsers: IE5.5, IE6
Description: This option controls whether the menu will write CSS properties for controlling the look of the scrollbar in IE5.5 and IE6. It's not enough to set any of the scrollbar options, you will have to explicitly set this variable to true for them to work.
