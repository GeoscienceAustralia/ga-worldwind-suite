MTMSubsGetPlus

Possible values: Never, Always or Submenu, as a string
Default value: "Submenu"
Requirements: None
First available: 2.3.0
Supported browsers: All
Description: This controls the +/- signs showing up in the menu. If set to Never there will never be any such signs. If it is set to Submenu a submenu will have a +/- sign next to it when it contains another submenu (this is similar to Windows Explorer®). Lastly you can set it to Always and all submenus will have the +/- sign. Please note that setting MTMEmulateWE to true will force this variable's value to Always, and that the value of this option is case-insensitive.
