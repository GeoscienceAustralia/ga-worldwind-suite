MTMLinkedInitFunction

Possible values: A valid function name as a string
Default value: Not set
Requirements: MTMLinkedJSURL must be set and obviously the function must exist in your linked JavaScript file
First available: 2.3.1
Supported browsers: All but IE4
Description: If you wish to call a function in your linked JavaScript file this variable enables you to do so. This way you can initialize variables or set up event handling in the menu frame.
