MTMUseCookies

Possible values: true or false
Default value: false
Requirements: none
First available: 2.3.0
Supported browsers: All
Description: This option controls whether the menu uses cookies. You will have to set it to true to enable storage of the menu expansion state. The ability to store the tracked item in a cookie is an extension of this feature, so it'll have to be set to true if you wish to use MTMTrackedCookieName to store the tracked item.
