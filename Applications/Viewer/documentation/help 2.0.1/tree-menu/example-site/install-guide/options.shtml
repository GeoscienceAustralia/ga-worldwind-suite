Setting options

These options used to be found in the file "code.html", but now that they all have default values the file doesn't contain any. Instead you'll have to insert the options that you need and make sure they have the right values. To help you on your way there's a list of default values so you know what they all are and also a sample code.html which will show they're inserted into the file.
