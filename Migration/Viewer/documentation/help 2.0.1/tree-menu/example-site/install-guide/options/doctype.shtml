MTMDOCTYPE

Possible values: A valid DOCTYPE as a string
Default value: Not set
Requirements: none
First available: 2.3.1
Supported browsers: IE4, NN3, NN4, Opera 5 & 6
Description: This variable defines the DOCTYPE written to the document in the menu frame by the menu code, and should therefore be set to a valid HTML DOCTYPE. It is not used by the DOM-capable browsers, they instead use any DOCTYPE defined in the document found in the frame where the menu is shown.
