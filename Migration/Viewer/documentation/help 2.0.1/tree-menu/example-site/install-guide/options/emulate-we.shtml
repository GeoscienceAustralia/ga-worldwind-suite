MTMEmulateWE

Possible values: true or false
Default value: false
Requirements: none
First available: 2.2.0
Supported browsers: All
Description: This variable controls whether the menu will emulate Windows Explorer® when a submenu has a URL attached to it. If it's set to true all submenus will have a +/- sign, and only the +/- sign will expand/collapse the submenu. The icon and the item's text will point to the associated document. Set it to false and the value of MTMSubsGetPlus will determine whether the +/- sign is shown or not. A collapsed submenu will then link to the associated document and also expand the submenu, while when it's open the +/- sign (if present) and the icon will collapse it and the item's text will reload the associated document.
