MTMLinkedSS

Possible values: true or false
Default value: false
Requirements: none
First available: 1.0.0
Supported browsers: All
Description: This variable controls whether the menu uses a linked style sheet (which is then specified with MTMSSHREF) or if it should use its own inline style sheet. If you set it to true all user-configurable font & colour settings will be ignored. For more information about the CSs properties used by the menu see CSS Properties.
