MTMCookieDays

Possible values: A number from 0 upwards
Default value: Not set
Requirements: MTMUseCookies = true;
and MTMCookieName must be set
First available: 2.3.0
Supported browsers: All
Description: This option controls how many days the cookies used by the menu will be stored. A value of 0 means they'll only last for the current browser session, and a value of 7 will mean "one week from the last visit".
